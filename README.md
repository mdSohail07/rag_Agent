# 🚀 RAG Agent

A simple Retrieval-Augmented Generation (RAG) based AI agent built using Python and LangChain.

## ✨ Features

- PDF document processing
- Semantic search
- AI-powered question answering
- Streaming responses
- Modular RAG architecture

## 🛠️ Technologies Used

- Python
- LangChain
- Gemini Embeddings
- Groq Llama 3.1
- Streamlit

## 📂 Main File

```python
apps/rag_agent.py
```

## ▶️ Run

```bash
streamlit run apps/1_qna_bot.py
```
